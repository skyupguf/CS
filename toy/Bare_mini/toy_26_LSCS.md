# **LSCS**

## **문제**
정수를 요소로 갖는 배열을 입력받아 다음의 조건을 만족하는 LSCS를 리턴하라

- `LSCS`: 주어진 배열의 연속된 부분 배열의 합을 구한다고 할 때, 이 중 가장 큰 값(Largest Sum of Contiguous Subarray)
- `연속된 부분 배열들`: 배열 `[1,2,3]`의 연속 부분 배열은 `[1]`, `[1, 2]`, `[1, 2, 3]`, `[2]`, `[2, 3]`, `[3]` 이다.

## **입력**

### **인자 1 : arr**
- `number` 타입을 요소로 갖는 배열
- `arr.length`는 60,000 이하
- `arr[i]`는 -100,000 이상 100,000 이하의 정수

## **출력**
- `number` 타입을 리턴해야 한다.

## **주의사항**
- 배열의 모든 요소가 음수인 경우도 있다.

## **입출력 예시**

```tsx
let output = LSCS([1, 2, 3]);
console.log(output); // --> 6

output = LSCS([1, 2, 3, -4]);
console.log(output); // --> 6 ([1, 2, 3])

LSCS([1, 2, 3, -4, 5]);
console.log(output); // --> 7 ([1, 2, 3, -4, 5])

LSCS([10, -11, 11]);
console.log(output); // --> 11 ([11])
```

## **Tip**
- LSCS를 계산하는 효율적인 알고리즘(`O(N)`)이 존재한다.
